PROBANDO GIT

CAMBIOS EN LA RAMA MASTER
Comando de Configuracion de Usuario git config --global user.name Configuramos el usuario
Comando de Configuracion de Email git config --global user.email Configuramos el mail
Comando de Inicio git init . Crea el repositorio Git
Comando de seguimiento git status Seguimiento de los archivos
Comando Agregar git add . Subimos todos los archivos
Comando Commit git commit -m "Mensaje de confirmacion" Comiteo los cambios
Comando Ver cambios git diff "nombre del archivo" Diferencias de los archivos
Comando Ver Todos los commit git log Listados de todas las diferenciass
Comando Rama del repositorio git branch Veo donde estoy parado
Comando Nueva Rama git branch nuevoNombre Creo un nuevo branch
Comando Nueva Rama git checkout nuevoNombre Me cambio de rama
Comando Mezclar Distintos Branch git merge nuevoNombre Mezcla codigo fuente con sus cambios
Comando Borrar una rama git branch -d nombreBranch Elimina una rama
Comando Borrar una rama con mayor permiso git branch -D nombreBranch Elimina una rama asi no haya hecho un merge con el master
Comando Crear y cambiar a una rama git checkout -b nombreBranch Crea una rama y me para en el repositorio de esa rama
Comando traer una commit especifico desde otro branch git cherry-pick identificadorDeEseCommit Trae ese cambio de ese branch.
Revertir un commit especifico git revert identificadorDeEseCommit revierte ese commit.
